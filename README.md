# AONYA

A top-down 2D game made in Unity for a school project. 

Trailer: https://youtu.be/cMPBU7mgRAo

---

Creators: Mustafa Al-Azzawe, Daniyal Shah

Controls:

> W,A,S,D - Movement <br>
> E - Inventory <br>
> I - Interact <br>
> P - Pause <br>
> [SPACE] - Attack
